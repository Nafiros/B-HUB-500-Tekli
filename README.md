## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Install:
* nodejs
Dependencies:
* react.js

### Installation

1. Clone the repo
```sh
git clone git@github.com:Nafiros/B-HUB-500-Tekli.git
```
2. Launch the program with python 3.8
```sh
npm install
```
3. Run the app
```sh
npm start
```
